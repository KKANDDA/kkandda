## 지금은 마크다운 문법을 배우고 있습니다.

"#"을 이용하면 글자 크기를 조절할 수 있고, "-, +, *"을 이용하면 넘버링 없는 문장을 시작할 수 있고, "1."를 이용하면 넘버링이 달린 문장을 쓸 수 있네요.

# 제일 큰 글자 "#"
###### 제일 작은 글자 "######"

- 넘버링이 없는 문장 "-"
1. 넘버링이 달린 문장 "1."

글을 아주 길게 쓰면 줄바꿈이 일어나죠, 하지만 임의로 줄바꿈을 줄려면 두 줄 이상을 줄바꿈을 주어야 합니다.


이렇게요. (엔터를 두 번 때렸습니다.)
